# n8n Automation Workflows 🚀

Welcome to my automation portfolio! This repository contains professional workflows built using **n8n**, focused on AI integration and process efficiency.

## 📂 Projects Overview

### 1. [AI News Summary](https://github.com/ayham-7asan/n8n_Projects/tree/main/Summary_News)
- **Goal:** Automatically fetch, summarize, and deliver daily AI news.
- **Tools:** n8n, OpenAI (GPT-4), RSS Feeds.

### 2. [Proposal Generator](https://github.com/ayham-7asan/n8n_Projects/tree/main/Generate_Proposal)
- **Goal:** Streamline the creation of business proposals using AI agents.
- **Tools:** n8n, AI Agents, Google Docs Integration.

### 3. [University Doctors Booking Agent](./University_Doctors_Booking)
- **Goal:** Automate academic office hour scheduling using AI to handle requests and conflict checks.
- **Tools:** n8n, AI Agents (GPT-4o-mini), Google Calendar, Google Sheets, Gmail.
